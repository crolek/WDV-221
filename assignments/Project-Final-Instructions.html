Project - Final Instructions

Overall Instructions

Do this project in parts. Work on one part at a time, make sure you meet the criteria around it, and then add the next part.

Part 1

The first part you will be creating a manual image slide show. The slide show component will start with 1 of 3 images on the page.
Example images are below, but you can use any images you'd like. (You have to use at least 3 images to start.)

  • https://crolek.github.io/WDV-221/assignments/imgs/final/garden.jpg
  • https://crolek.github.io/WDV-221/assignments/imgs/final/stewie.jpg
  • https://crolek.github.io/WDV-221/assignments/imgs/final/lilsquiggle.jpg

These goal of this part is to rotate through each image in order. When the user clicks the Next button it should switch-out or hide the current image and show the next image. If the final image is displayed, and the user clicks the Next button it should show the first element in the list. This same concept applies applied to the Previous button, but respectively going in reverse.

Part 2

In this part you will be adding the ability to add a new image to the list. Treat the value that is added to the "New Image" field as a URL. An example of an image URL that might get added is this:
http://pbs.twimg.com/profile_images/554344612832092160/WRLmixCH_400x400.png

Part 3

Add validation to the "New Image" field that makes sure the feild isn't empty. If it is empty then display an error.
